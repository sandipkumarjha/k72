
# 🚀 K72 Website Clone (React)

A modern and interactive clone of the K72 website built using React.
This project focuses on smooth animations, clean UI, and responsive design to enhance frontend development skills.

---

## 🔥 Features

* ⚡ Smooth animations and transitions
* 🎨 Modern UI/UX design
* 📱 Fully responsive (mobile + desktop)
* 🧩 Component-based architecture using React
* 🚀 Optimized performance

---

## 🛠️ Tech Stack

* React.js
* HTML5
* Tailwind CSS
* JavaScript (ES6+)
* GSAP (for animations)

## ⚙️ Installation & Setup

1. Clone the repository

```
git clone https://github.com/your-username/k72-clone.git
```

2. Navigate to project folder

```
cd k72-clone
```

3. Install dependencies

```
npm install
```

4. Run the development server

```
npm run dev
```
